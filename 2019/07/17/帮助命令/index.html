Linux 自带的帮助文档是最准确、最权威可靠的资料。

man 命令

项目 内容
命令名称 man
英文原意 format and display the on-line manual pages
所在路径 /usr/bin/man
执行权限 所有用户
作用 显示联机帮助手册

info 命令

info 命令也可以获取帮助，和 man 不同的是，info 命令的帮助信息是一套完整的资料，每个单独的帮助信息只是这个资料的某一小章节。

  • info 命令的快捷键
    快捷键 作用
    ↑ 向上移动一行
    ↓ 向下移动一行
    Pg Up 向上翻一页
    PgDn 向下翻一页
    Tab 在有 “*” 的节点间进行切换
    Enter 进入有 “*” 的子页面，查看详细帮助
    u 进入上一层信息
    n 进入下一小节信息
    p 进入上一小节信息4
    ? 查看帮助信息
    q 退出

help 命令

该命令只能查看 shell 内置命令的帮助，完全可以用 man 替代，并不经常使用。‘’
